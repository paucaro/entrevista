# Mi Entrevista

A new Flutter M-learning to job interview.

## Getting Started

This project is a Flutter application build with firebase.

To run this project
- Install all packages
```
flutter pub get
```
- Configure google-services from [firebase console](http://console.firebase.google.com/)
